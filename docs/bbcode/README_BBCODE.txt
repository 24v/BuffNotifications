[size=6][b]Buff Notifications[/b][/size]

A Stardew Valley mod that displays notifications when buffs start, are about to expire, and end.

[img]https://raw.githubusercontent.com/24v/BuffNotifications/main/docs/images/Buffs_Replaced.png[/img]

[size=5][b]Features[/b][/size]

[*] [b]Buff Start Notifications[/b]: Receive a notification when a buff is activated
[*] [b]Buff Expiring Warnings[/b]: Get a warning when a buff is about to expire (10 seconds before expiration -- configurable)
[*] [b]Buff End Notifications[/b]: Be notified when a buff has ended

[size=5][b]Installation[/b][/size]

[*] Install [url=https://smapi.io/]SMAPI[/url]
[*] Download the latest version of Buff Notifications from [url=https://www.nexusmods.com/]Nexus Mods[/url] or the [url=https://github.com/24v/BuffNotifications/releases]GitHub releases page[/url]
[*] Extract the downloaded file to your Stardew Valley Mods folder
[*] Run the game using SMAPI

[size=5][b]Configuration[/b][/size]

The mod can be configured using the config.json file or through Generic Mod Config Menu if you have it installed.

Options include:
[*] Enable/disable buff start notifications
[*] Enable/disable buff expiring warnings
[*] Enable/disable buff end notifications
[*] Adjust warning time before buff expiration

[size=5][b]Compatibility[/b][/size]

[*] Stardew Valley 1.6.14 or later
[*] SMAPI 4.1.7 or later
